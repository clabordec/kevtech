<h1>Installing and setting up Windows 10 Pro in VMware Workstation Pro</h1>
This project outlines adding RSAT tools to a client machine along with adding a computer to the domain with that same client machine.<br />


<h2>Environments and Technologies Used</h2>

- VMWare Workstation Pro (Virtual Machines/Compute)

<h2>Operating Systems Used </h2>

- Windows 10 Pro

<h2>High-Level Deployment and Configuration Steps</h2>

- Download the following RSAT tools:
    - RSAT: Active Directory Certificate Services ToolsActive Directory Certificate Services Tools
    - RSAT: Active Directory Domain Services and Lightweight Directory Services Tools
    - RSAT: DHCP Server Tools
    - RSAT: DNS Server Tools
    - RSAT: Group Policy Management Tools
    - RSAT: Remote Desktop Services Tools
- Set the IP Address for `Client02` along with making the Preferred DNS Server match the IP Address of the domain controller
    - IP Address: 10.85.206.11
    - Subnet Mask: 255.0.0.0
    - Default Gateway: 10.85.206.1
    - Preferred DNS server: 10.85.206.5
    - Alternative DNS server: 127.0.0.1
- Add `Client02` to the domain by inputting the `Help Desk` user credentials
- Verify that the computer was added to the domain
